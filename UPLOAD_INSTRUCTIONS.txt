HOSTINGER UPLOAD GUIDE - MANUAL FOLDER UPLOAD
==============================================

STEP 1: Delete everything in public_html
- Select all files and folders
- Delete them

STEP 2: Upload folders one by one (in this order):

1. Upload the _next folder:
   - In Hostinger File Manager, click "New folder" 
   - Name it: _next
   - Open the _next folder
   - Upload ALL contents from: out/_next/
   - This includes: static/ folder and other folders inside _next

2. Upload the images folder:
   - Create folder: images
   - Upload all PNG files from: out/images/

3. Upload the videos folder:
   - Create folder: videos  
   - Upload both .mov files from: out/videos/

4. Upload the services folder:
   - Create folder: services
   - Upload all HTML and TXT files from: out/services/

STEP 3: Upload root files (directly to public_html):
- index.html
- about.html
- contact.html
- services.html
- privacy.html
- cookies.html
- 404.html
- sitemap.xml
- robots.txt
- icon.svg
- All .txt files (about.txt, contact.txt, etc.)

CRITICAL: Make sure folders are created FIRST, then files uploaded INTO them.
